Account Chart Flat

Enhancements for a better Odoo accounting experience.

Once installed your chart of accounts will be flat. No more parent_left, parent_right and parent_id.

This speeds up accounting a lot and is the only way to do accouting in Odoo if you have big charts of accounts.

Installation

Install as you would install any Odoo addon.

Dependencies

This addon requires Odoo accounting.

Configuration

This addon does not require any setting.

Usage

Nothing particular to use this addon - just install it.
