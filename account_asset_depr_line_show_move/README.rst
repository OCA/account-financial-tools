==============================================================
Button for showing the move on account asset depreciation line
==============================================================

This module adds a button to show the journal entry associated with an asset
depreciation line.

Usage
=====

On each depreciation line that has created a journal entry, you have a new
button with an arrow that allows you to navigate to the journal entry.
