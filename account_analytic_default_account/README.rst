Module name
===========

This module was written to extend the functionality of paying an invoice
and allow you to get a default analytic account on counterpart move line
when invoice's currency is not the same than company currency

Installation
============

To install this module, you need to:

* just install it !

Configuration
=============

To configure this module, you need to:

* go to Accounting / Analytic Accounting / Analytic Defaults
* create analytic defaults and set an general account in it

Usage
=====

To use this module, you need to:

* Nothing

For further information, please visit:

* https://www.odoo.com/forum/help-1
